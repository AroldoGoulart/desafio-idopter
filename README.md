Esse repositorio é dedicado ao desafio tecnico para ingresso part-time na IdopterLabs.com

## Como rodar

- Baixe o repositorio
```
git clone https://github.com/SnowLew/desafio-idopter.git

```
yarn
```

- Seja feliz :)
```
npx react-native run-android
